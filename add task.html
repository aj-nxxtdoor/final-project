About StudyTrack

A simple task manager built to help students stay organized.

What is StudyTrack?

StudyTrack is a lightweight study planner web app that helps you keep track of assignments, projects, and any other academic tasks. Everything is saved in your browser's local storage, so your data stays with you even after you refresh the page.

You can add tasks with a deadline, priority level, and notes — then manage them from your dashboard. Mark tasks as done, delete ones you no longer need, and filter by priority or status.

Why I Built This

This was built as a final project for a web development course. The goal was to practise using HTML, CSS, and vanilla JavaScript together — specifically DOM manipulation, event handling, form validation, and localStorage.

I chose a task manager because it covers all these requirements in a way that's actually useful day-to-day as a student.

Technologies Used

  • HTML5 — semantic markup and page structure
  • CSS3 — Flexbox layout, media queries, custom properties
  • JavaScript (Vanilla) — DOM manipulation, localStorage, form validation

Known Issues / Limitations

  • Data is only stored locally — no backend or account system
  • Clearing browser data will delete all tasks
  • No drag-and-drop reordering yet (maybe a future feature)
